# Security

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

1. **Do not** open a public GitHub issue for security vulnerabilities.
2. Go to the **Security** tab of this repository and use "Report a vulnerability" to submit a private advisory, or contact the maintainers via the email listed in the repository description.
3. Include a description of the vulnerability, steps to reproduce, and potential impact.
4. Allow a reasonable time for a fix before disclosing publicly.

We appreciate your efforts to disclose your findings responsibly.
